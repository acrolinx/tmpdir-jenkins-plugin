Creates a private temporary directory before each build and deletes it afterwards in order to keep the system-wide temporary directory tidy.

The environment variables TMP, TEMP and TMPDIR are also exported so that all sane utilities should automatically use this temporary directory.

Java is not one of those sane applications (at least on UNIX/Linux), so the Java system property java.io.tmpdir is set using the environment variable JAVA_TOOL_OPTIONS. Should this variable already be set, then this property is appended to the existing value of that variable.
