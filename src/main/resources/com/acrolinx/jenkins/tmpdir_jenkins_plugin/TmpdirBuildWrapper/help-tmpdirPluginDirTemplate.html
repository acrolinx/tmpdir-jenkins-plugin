Path template to use when creating a TMPDIR for a job.

May contain the usual Jenkins environment/build variables. If relative, then it is made absolute by prepending the value of the java.io.tmpdir Java system property on the node that is used to execute a particular build.

For example, ${BUILD_TAG}-tmp may end up being something like /var/tmp/foobar-123-tmp if java.io.tmpdir on the node is set to /var/tmp/.
